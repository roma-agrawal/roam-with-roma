# 🇨🇭 Roam with Roma v1.0

A personal Progressive Web App (PWA) created for Roma's Switzerland Trip (17–25 August 2026).

---

## Features

- 🇨🇭 Day-wise itinerary
- 🪂 Beautiful timeline view
- ✈ Flight details
- 🏨 Hotel information
- 💰 Budget
- 🎒 Packing checklist
- 📞 Emergency contacts
- 📱 Installable PWA
- 🌙 Dark Mode
- ⚡ Works Offline

3. Open
